## Building
```
git clone https://github.com/notmyst33d/datacenter.ts
npm i
npm run bundle
```

You should get dist/datacenter.mjs as the output.

## Developing
```
git clone https://github.com/notmyst33d/datacenter.ts
npm i
npm run debug
```

This will run in monitor mode and will watch for source code changes.
