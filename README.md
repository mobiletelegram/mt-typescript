## Building
```
git clone https://github.com/mobiletelegram/mt
npm i
npm run bundle
```

You should get dist/mt.mjs as the output.

## Developing
```
git clone https://github.com/mobiletelegram/mt
npm i
npm run debug
```

This will run in monitor mode and will watch for source code changes.
